Download the installer and run Just Share Setup.exe. Follow the steps to choose your install location and preferences.

  • Supports Windows 10/11
  • Creates desktop and Start menu shortcut
  • No account or setup needed

❓ FAQ

Is my data shared online?

Nope. All transfers are local unless ngrok tunneling is manually enabled.

What if I close the tray?

The app stays running in the background. Right-click the tray icon to quit completely.

Where do uploaded files go?

They are stored in the uploads/ folder alongside the app.
